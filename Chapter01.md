# 1️⃣ What is Java?

## English

Java is a **High-Level, Object-Oriented, Platform-Independent Programming Language** used to develop desktop, web, mobile, enterprise and cloud applications.

Java follows the principle:

> **Write Once, Run Anywhere (WORA)**

because Java programs can run on any operating system that has a JVM.

## 🇮🇳 Hindi

Java ek **High-Level, Object-Oriented aur Platform-Independent Programming Language** hai.

Java ki help se hum

- Desktop Applications
- Web Applications
- Android Apps
- Enterprise Software
- Cloud Applications

develop kar sakte hain.

Java ka famous principle hai:

> **Write Once, Run Anywhere (WORA)**

# Definition

Java is a high-level, object-oriented, platform-independent programming language developed by **James Gosling** in **1995**.

# 3️⃣ Why is Java Called Java?

# English
Initially the language was named **Oak**.
Since the name Oak was already registered, the development team selected the name **Java**, inspired by Java Coffee.

#  Hindi
Sabse pehle Java ka naam **Oak** tha.
Baad me pata chala ki Oak naam pehle se registered hai.
Isliye team ne iska naam **Java** rakh diya.

# 4️⃣ Features of Java

## 1. Simple
Java syntax is easy to learn.
(Java ka syntax simple aur beginner friendly hai.)

## 2. Object-Oriented
Everything in Java revolves around Classes and Objects.
(Java OOPs concepts follow karta hai.)

## 3. Platform Independent
Java programs run on every operating system.
Reason: JVM

## 4. Secure
Java provides
- Bytecode Verification
- JVM Security
- No Pointer Arithmetic

## 5. Robust
Java is reliable because of
- Exception Handling
- Garbage Collection

## 6. Multithreaded
Java allows multiple tasks to execute simultaneously.

Example:
- Video Playing
- Downloading
- Comments Loading

## 7. Portable
Java programs can be moved easily from one platform to another.

## 8. High Performance
Java uses
> JIT (Just In Time Compiler) to improve performance.

## 9. Distributed
Java supports network-based applications.

## 10. Dynamic
Java loads classes dynamically during runtime.
